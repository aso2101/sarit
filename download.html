SARIT: Text Archive Downloads.

All texts accessible through the SARIT web interface -- Basic Search and Advanced Search -- are available for download.

SARIT PDF and HTML downloadable texts have been produced automatically from XML files. The web interface can help with complex word searches, word counts and collocations and so on. The downloadable HTML files could be used for local and off-line searches, while the PostScript and PDF files could be printed for personal or class use.

If you download the XML TEI P5 file, you can use a convenient service such as OxGarage to convert it to another format that is useful to you, for example for editing locally, or reading on an ebook like the Kindle. OxGarage supports the following output formats, amongst others: ePub, LaTeX, Microsoft Word (.doc and .docx), LibreOffice/OpenOffice (.ods, .sxc, .sxw, .odt), PDF, Plain text, Rich Text Format (.rtf), and xHTML.
